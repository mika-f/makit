# Makit brand

Vector source files for the Makit mark, primary lockup, and monochrome lockup.

Open `index.html` in a browser to view the Tailwind CSS brand reference page. It uses the official Tailwind browser CDN, so an internet connection is required for the reference page's styles; the SVG files themselves are fully standalone.

## Assets

- `src/makit-mark.svg` — standalone icon
- `src/makit-lockup.svg` — icon, wordmark, and tagline
- `src/makit-monochrome.svg` — one-colour icon and wordmark; set its CSS `color` to change the ink colour
